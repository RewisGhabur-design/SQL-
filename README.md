# Customer Churn Analysis

## Overview
This project analyzes customer churn trends using transactional and behavioral data. It identifies key drivers of churn and visualizes retention metrics to support strategic decision-making.

## Tools Used
- Python (Pandas, NumPy, Matplotlib)
- SQL Server
- Power BI

## Key Features
- Data cleaning and transformation
- Churn rate calculation
- KPI dashboard with filters for region and customer segment
- Scenario-based insights for retention strategies

## Insights
- High churn observed in Segment B during Q2
- Late payments and low engagement are top predictors of churn
